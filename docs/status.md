# Purpose

This repository provides a reusable ROAMS starter template for running software projects through role-based orchestration, selective memory, skill-driven workflows, and lightweight bootstrap guidance.

# Current Stage

The project is in an early post-v1 stage with the core operating model, role skills, research capability, roadmap governance, Phase 4 subagent orchestration, and Phase 5 bootstrap automation completed.

# Near-Term Goals

The next steps are to expand the evaluation harness, keep validating the bootstrap workflow against real repo setups, and add new process only where repeated usage proves it is helpful.

# Open Questions

The main open questions are how far the verification harness should go before it becomes cumbersome and which repeated bootstrap patterns, if any, justify deeper automation later.
