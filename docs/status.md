# Purpose

This repository provides a reusable ROAMS starter template for running software projects through role-based orchestration, selective memory, and skill-driven workflows.

# Current Stage

The project is in an early post-v1 stage with the core operating model, role skills, research capability, roadmap governance, and Phase 4 subagent orchestration completed.

# Near-Term Goals

The next steps are to implement Phase 5 bootstrap automation, continue refining the workflow through verification, and expand the template only where repeated usage proves it is helpful.

# Open Questions

The main open questions are which setup actions deserve reusable automation support first and how much bootstrap help is useful before the template becomes overbuilt.
